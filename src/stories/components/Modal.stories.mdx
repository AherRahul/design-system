# Modal

### Modals are like custom dialogs, windows that display additional content on a layer above the current page, with an overlay covering the page and making it temporarily inaccessible.
---
<br />

## When to use:
- It is necessary to notify the user of some process;
- Decision-making is necessary to complete an action;
- If you want to carry out action within a page parallel to its main flow;

<br />

## When not to use:
- It is necessary to display a large amount of information or complex flows;
- It must be used in conjunction with other modals and popovers (cascade).
