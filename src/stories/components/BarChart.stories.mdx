# BarChart

### A reusable chart component to display data visually.
---
<br />

## When to use:
- Display data in visual formats;
- It is effective when displaying many categories;
- View trends, proportions or comparisons in data.

<br />

## When not to use:
- For simple information that can be displayed in a tabular or text format;
- It is not recommended for percentage comparison or parts of a whole.
