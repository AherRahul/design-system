# Table

### Tables are components used to display and organize content, facilitating the comparison and analysis of information.
---
<br />

## When to use:
- Want to compare data.
- It is necessary to display data in a tabulated form.

<br />

## When not to use:
- It is necessary to interact/perform an action with the content presented in the table.
