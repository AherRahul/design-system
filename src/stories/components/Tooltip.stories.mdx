# Tooltip

### Tooltips are non-interactive floating labels used to explain an interface element or feature. They can be activated with hover.
---
<br />

## When to use:
- The text to be displayed as a label is concise, with a maximum of 4 words.
- To display supplementary information about a feature or interface element.


<br />

## When not to use:
- To communicate errors and system status.
- To show critical information for understanding a feature or interface element.
