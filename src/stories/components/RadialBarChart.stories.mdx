# RadialBarChart

### A RadialBarChart is a variation of the BarChart plotted in a polar rather than Cartesian coordinate system.
---
<br />


## When to use:
- Display data with graphic appeal;
- Compare data;


<br />

## When not to use:
- When high fidelity is required in the display of information, with the aim of making accurate comparisons between data;
- When the data presents categories or subcategories;

---

<br />


## Preview

By sending an array of colors to the colors prop, the RadialBarChart colors will be defined
from this array, with each segment of the graph corresponding to the defined color
in the corresponding position of the array.
