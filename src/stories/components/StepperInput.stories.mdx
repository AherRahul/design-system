# StepperInput

### StepperInputs are numeric inputs with increment and decrement buttons always visible.

---
<br />

## When to use:
- If you need an input of type number.
- The number to be added to StepperInput is small or has a few digits.

<br />

## When not to use:
- The data to be used in the component is not numeric.
- The data to be used in the component is too large.
- The data to be used in the component is not incremental.
