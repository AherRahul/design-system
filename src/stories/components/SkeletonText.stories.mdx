# Skeleton Text

### Skeleton texts allow displaying a loading state for various component types
### while your data is being fetched.
---
<br />

## When to use:
- Inform the user that the component information is being loaded.
- The content to be loaded is dynamic.

<br />

## When not to use:
- The content to be loaded is static.

## Comments
- The Skeleton text must be adjusted according to the component to be used.
