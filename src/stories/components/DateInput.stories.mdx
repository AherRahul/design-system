# DateInput

### DateInputs are components for choosing dates and times within forms.
---
<br />

## We recommend using DateInputs when:
- There is a need to offer date options within a group of inputs on a form.

<br />

## We do not recommend using DateInputs when:
- It is necessary to display additional conditional information in the calendar (e.g. dates that are already occupied).
- The calendar must be displayed constantly.
