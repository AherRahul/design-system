# LineChart

### A reusable chart component to display data visually.
---
<br />

## When to use:
- Display data in visual formats;
- Ideal for showing trends over time;
- Comparison of multiple series.

<br />

## When not to use:
- For simple information that can be displayed in a tabular or text format;
- Discrete data without logical sequence;
- Large amount of data series.
