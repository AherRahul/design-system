# EmptyState

### EmptyStates are components used in instructive moments for users.

---
<br />

## When to use:
- It is necessary to instruct the user on how a particular page works.
- A page or component does not have default content to be displayed.
- A certain action results in an error.

<br />

## When not to use:
- Used as a loader.
- Used only with title.
