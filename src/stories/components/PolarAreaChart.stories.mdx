# PolarAreaChart

### A reusable chart component to display data visually.
---
<br />

## When to use:
- Display data in visual formats;
- View trends along continuous variables such as directions or degrees.

<br />

## When not to use:
- For simple information that can be displayed in a tabular or text format;
- When the data emphasizes the numerical comparison of quantities;
- Not recommended for representing time series or discrete data.
