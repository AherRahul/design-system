# MobileNavigation

### MobileNavigation is a sidebar that provides quick and easy access to the main sections of the application in the mobile environment.
---
<br />

## When to use:
- When navigation in the mobile version of the application is necessary;
- When it is necessary to facilitate access to different sections of the application, ensuring that the user can easily navigate between pages without getting lost;
- When vertical space is limited, making the most of the mobile device's screen width.

<br />

## When not to use:
- When in a desktop environment, where a top or side navigation bar may be more appropriate;
- When on screens that require significant horizontal space, as the menu may overlap or obstruct the main content.
