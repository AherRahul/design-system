# Timeline

### Timelines are components used to present information in chronological order.

---
<br />

## When to use:
- If there is a set of information that can be categorized by date.


<br />

## When not to use:
- If you need a stepper component.
- The order of the information you are presenting is irrelevant.

<br />

## Comments:
- The component `Timeline` is a wrapper and is used to group a series of components `TimelineItems` into the same component via slots. For more information, access the component documentation.
