# AvatarGroup

### AvatarGroups display multiple avatars grouped together in a stack.
---
<br />

## When to use:
- Use AvatarGroups when, in a given context, several users share responsibilities for 
information or business rules.

## When not to use:
- In cases where only one user's information will be presented.

## Comments:
- When you hover the mouse over each Avatar, a tooltip will be displayed with the user's 
name, if it has been entered.
- When the number of avatars was greater than the value specified in the maxCount prop, 
an avatar will be displayed with a counter indicating the number of avatars that were 
deleted from the AvatarGroup. This counter is clickable and opens a Popover with the 
list of suppressed avatars.
