# Button

### Buttons are components that allow the user to perform an action with one touch.
---
<br />

## When to use:
- It is necessary to communicate to the user that they can perform an action in the interface, whether in dialogs, modal windows, forms, cards, etc.


<br />

## When not to use:
- It is necessary to give the option to perform an action as a small detail in the interface. In this case, we recommend using links.
