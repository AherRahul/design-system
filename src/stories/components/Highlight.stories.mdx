# Highlight

### Highlights are components used to highlight important words or phrases in a given context.
---
<br />

## When to use:
- If you want to guide the user's attention to specific words or phrases in the interface 
(in these scenarios, animated Highlights can be even more effective than regular ones).
- You are implementing a diff feature and need to indicate elements that have been replaced by others.

<br />

## When not to use:
- If you want to create hierarchy on the page. For these cases, use typographic hierarchy mechanisms such as font weight, size, or color.
- The element that the Highlight will be wrapped around is not a text element.
- Very large sections of text are highlighted. This can end up resulting in the opposite of what you expect with the use of Highlights.
