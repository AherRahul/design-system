# RadioButtonGroup

### RadioButtonsGroup are custom radio buttons for easy selection and feedback.
---
<br />

## When to use:
- The options that the component seeks to provide are mutually exclusive.
- It is necessary to use a ```<input type="radio"``` />.

<br />

## When not to use:
- Two or more options can be selected at the same time.
- There are 4 or more options. Prefer Dropdowns in these cases.

## Note:
- You can disable each of the radio buttons by passing the ```disable: true``` property in the respective object
in the options array. If you want to disable the entire group, you can set the ```disabled``` prop to true in the component.

<br />
