# FloatingAssistant

### Floating Assistant is a floating component that can be used to provide usability tips or specific information about page content.
---
<br />

## When to use:
- There is a need to provide tips or additional information about a certain element or section of the page.
- It is necessary to direct the user to an external page that contains additional information/tutorials about the content.
- The objective is to reduce the amount of information displayed on the screen, keeping the user's focus on the relevant element.


<br />

## When not to use:
- The component becomes intrusive and disrupts navigation and user experience.
- There is not enough space to display the component and it could be confused with the page content.
