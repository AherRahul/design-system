# Radio

### Radio allows users to select an option from a set.
---
<br />

## When to use:
- The options that the component seeks to provide are mutually exclusive.

<br />

## When not to use:
- Two or more options can be selected at the same time.
- There are 4 or more options. You can use the component `Dropdown` these cases.

<br />
