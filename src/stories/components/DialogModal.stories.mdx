# DialogModal

### DialogModal is a component used to interact with the user in critical or informative situations.
---
<br />

## When to use:
- When it is necessary to obtain user confirmation for actions that may have significant consequences, such as deleting an item;
- When an error occurs in the system and the user needs to be notified;
- When it is necessary to inform the user about something important, but which does not require immediate action.

<br />

## When not to use:
- When the messages to be passed to the user are trivial and do not require the user's immediate attention. In these cases, consider using notifications or alerts.
