# Link

### Links are components used to navigate to another page and can open a new tab.
---
<br />

## When to use:
- It is necessary to navigate to another section or page.
- It is necessary to redirect to an external website or document.

<br />

## When not to use:
- An action needs to be performed (example: save, edit). In these cases, it is recommended to use the Button component.

<br />
