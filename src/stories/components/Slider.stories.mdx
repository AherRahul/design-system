# Slider

### Sliders are components used to select values ​​within a range.
---
<br />

## When to use:
- We recommend using sliders when it is necessary to select values ​​within a range,
especially when it is necessary to select more than one value.
- We recommend using sliders when the speed of selecting values ​​in a range is
more relevant than precision.

<br />

## When not to use:
- We do not recommend the use of sliders when there is no need for great precision in selecting values.
- We do not recommend using sliders in conjunction with inputs so that the user can edit
given by both the slider and the inputs.

## Comments:
- The RDS slider is a wrapper of the <a target="_blank" href="https://nightcatsama.github.io/vue-slider-component/#/">Vue slider component</a>
with its compliance with our Principles and recommendations. For detailed component information, we recommend
reading the documentation.
