# Select

### Selects allow the user to choose an option from a list of items.
---
<br />

## When to use:
- The user has to choose a single option from a list of several items.


<br />

## When not to use:
- The list of options has 3 items or less. In these cases, we recommend using radio buttons.
- The user needs to select more than one item from the list. In this case, use <a href="https://rds-design-system.netlify.app/?path=/docs/componentes-forms-clusteredmultiselect--clustered-multiselect">ClusteredMultiselects</a>.

<br />

## Comments:
- Place the options most likely to be selected at the top of the select. If in doubt, sort them in alphanumeric order.
- Select labels must be succinct (one to three words).
- Labels must be written with the first letter capitalized and the rest lowercase.
- ‚ö†Ô∏è **Important:** For the select to work correctly, the property whose values ‚Äã‚Äãwill be displayed in the select must be called `value`, as indicated in the example below in the props table.
