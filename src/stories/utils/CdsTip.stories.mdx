# CdsTip()

CdsTip() is an RDS directive used to implement tooltips. Hovering on the component triggers the display of the tooptip. To stop tooptip from being displayed, simply send null as text.

<hr />
<br />

#### Arguments

**(String):**
- Specifies the tooltip text
- Mandatory: *Yes*
- Valid values: any string

<br />

#### Return

None

<br />

#### Note:
- Uses [tippy.js](https://tippyjs.bootcss.com/) as a dependency. Reading the tippy documentation may help with using this directive.
