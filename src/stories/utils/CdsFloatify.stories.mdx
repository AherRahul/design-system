# CdsFloatify()

CdsFloatify() is an RDS directive that allows elements that use it to become floats.

<hr />
<br />

#### Arguments

**(String):**
- Indicates the positioning of the component in reference to the target element. It is passed as an argument to the directive, in the format `v-cds-floatify:[argument]`
- Mandatory: *No*
- Valid values: 'top', 'top-start', 'top-end, 'bottom', 'bottom-start, 'bottom-end, 'left', 'left-start, 'left-end, 'right' , 'right-start, 'right-end;

**(String):**
- The id of the element that ContrastChecker will use as a reference to position itself. It is passed as a value to the directive, in the format `v-cds-floatify="argument"`
- Mandatory: *Yes*
- Valid values: any string that indicates the id of a DOM element;

<br />

#### Return

None

<br />

#### Note:
- Uses [popper.js](https://popper.js.org/docs/v2/) as a dependency. Reading the popper documentation may help with using this directive.
