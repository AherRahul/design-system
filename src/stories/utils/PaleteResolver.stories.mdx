# PaleteResolver()

Mixin sass used to create classes with colors from the Design Dystem palette.

<hr />
<br />

PaleteResolver, similar to VariantResolver, implements a loop through RDS color tokens,
but while VariantResolver cycles through the palette returning specific shades, PaleteResolver cycles through colors
from the palette returning all available shade tokens, providing a single variable called `$color`.

<br />

#### Examples

<Story name="PaleteResolver">
</Story>

<Source
	language='css'
	format={true}
	code={/*css*/ `
.color-picker {
	&__swatch {
		@include paletteResolver using ($color) {
			background-color: $color;
			transition: $interaction;
		}
	}
}
`}
/>

The code above without would be the same as:
