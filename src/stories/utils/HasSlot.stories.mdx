# HasSlot()

Method used to check whether a specific slot is being used in the component.

<hr />
<br />

⚠️ *Method only available for internal development in RDS*

<br />

#### Arguments

**(Object):**
- Slots object obtained with `this.$slots`
- Mandatory: *Yes*
- Valid values: component slots object;

**(String):**
- Name of the slot you want to check usage
- Mandatory: *Yes*
- Valid values: any string containing the name of one of the component's slots;

<br />

#### Return

**(Boolean):** Indicates whether a specific slot is in use.
