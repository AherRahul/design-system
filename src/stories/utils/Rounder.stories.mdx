# Rounder()

Method used to determine the `border-radius` of an element based on its width.

<hr />
<br />

⚠️ *Method only available for internal development in RDS*

<br />

#### Arguments

**(Number):**
- Width of an element in the DOM
- Mandatory: *Yes*
- Valid values: any natural number;

**(Number):**
- Border-radius limiter. This prop limits the maximum value that can be returned.
- Mandatory: *No*
- Valid values: any natural number;
- Default value: 24px;

<br />

#### Return

**(Number):** Value to be used as border-radius, calculated from the width of an element in the DOM.
